Raptor FC 6RT Specifications

Main Features

  • Main Processor: i.MXRT1176 Processor with Cortex-M7 @ 1GHz & Cortex-M4 @ 400MHz
  • Accel + Gyro: BMI088, ICM-42688-P, ICM-42686-P
  • Magnetometer: Bosch BMM350
  • Barometer: Bosch BMP390, BMP388
  • Security: NXP EdgeLock SE051C2

Power

  • Operating Voltage: 4.75V ~ 5.7V
  • USB Power Input: 4.75V ~ 5.25V

Interfaces

  • SDHC interface
  • 12-Channel PWM
  • 8x UART
  • 4x I2C
  • 6x SPI
  • 3x CAN-FD
  • RMII 100M Ethernet
  • USB

Software

Open Source PX4 / NUTTX RTOS

Raptor CAN GPS Specifications

Main Features

  • GNSS Receiver: u-blox M9N
  • Processor: STM32F412CEU6 MCU (32-bit Arm® Cortex®-M4 @ 100MHz, 512KB Flash, 256KB RAM)
  • Magnetometer: Bosch BMM350
  • Barometer: Bosch BMP388
  • IMU: ICM-42688-P 6-Axis IMU
  • Safety Switch: Physical Button
  • Buzzer: Yes
  • Protocol: DroneCAN

Indicators

  • Safety LED
  • GPS Fix
  • RGB for System Status

Power & Physical

  • Power: 5V, 120mA Avg, 130mA Max
  • Weight: 33g
  • Make: Made in India

Connectors

  • 2x4 Pin JST GH CAN (Pixhawk Std)
  • 6 Pin JST SH (Pixhawk Std debug connector)

Raptor RTK GPS Specifications

Description

DroneCAN RTK GPS module with integrated Magnetometer, Barometer, IMU, Buzzer, and Safety Switch. Supports DroneCAN firmware updating for maximum compatibility.

Sensors

  • Ublox F9P GPS: L1/L2 multi-band GNSS receiver delivering centimeter-level accuracy in seconds. Concurrent reception of GPS, GLONASS, Galileo, and BeiDou. Multi-band RTK with fast convergence and high reliability.
  • Bosch BMM350: Magnetometer
  • Bosch BMP388: Barometer
  • Invensense ICM-42688-P: 6-Axis IMU
  • MCU: STM32

Features

  • Safety Button and Buzzer
  • Two Pixhawk Standard CAN Connectors (4-Pin JST GH)
  • F9P "UART 2" Connector (3-Pin JST GH - TX, RX, GND)
  • Pixhawk Standard Debug Connector (6-Pin JST SH)

LED Indicators

  • Safety LED
  • GPS Fix
  • RTK Status
  • RGB System Status

Power Requirements

5V, 170mA average, 180mA max

Includes

  • 4-Pin Pixhawk Standard CAN Cable
  • GPS Antenna
  • Ublox ANN-MB-00-00

Raptor SmartPDB Specifications

Main Features

  • Input Voltage: Up to 60V (14S battery)
  • Current Handling: Up to 400A
  • Supports 12 pairs of pads for powering up to 12 motors
  • Integrated voltage and current sensors
  • Integrated DC-DC converter from 10–60V to 5V @ 5A
  • Integrated DC-DC converter from 10–60V to 12V @ 5A
  • 5V and 12V power output terminals (standard 2.54mm connectors)
  • Buzzer
